# rs_toio_cube

Access test to toio core cube with Rust on Windows

## Getting Started

### Prerequisites

You pair toio core cube(s) with your PC before running this sample code.

### How to run

```
git clone https://github.com/kaz399/rs_toio_cube.git 
cd rs_toio_cube
cargo run --example connect
```
